# School Management System

Plateforme de gestion scolaire complète — projet DevOps & Dev.

## Stack
- **Backend** : Node.js 24 / Express / PostgreSQL / Redis
- **Frontend** : React 18 / Vite / Tailwind CSS
- **DevOps** : Docker / GitHub Actions / Terraform / Kubernetes + Helm

## Structure
\`\`\`
school-mgmt/
├── backend/        # API Node.js/Express
├── frontend/       # React/Vite
├── infra/          # Docker, Terraform, Kubernetes
└── .github/        # CI/CD GitHub Actions
\`\`\`

## Lancer le projet
\`\`\`bash
cd infra/docker
docker compose up -d
\`\`\`
